# Spootify 곡리뷰 서비스

## 기술스택

- Nodejs(Express)
- mongoDB
- JQuery
- bootstrap
- AWS EC2

## 주요기능

### 1. 리뷰작성
- 게시글 작성
- 작성한 값 검사

### 2. 메인화면
- 작성리뷰 목록 정렬
- 제목 클릭 시 상세페이지 이동

### 3. 상세페이지
- 게시글 상세 정보 표시
- 수정/삭제 버튼 시 페이지 이동

### 4. 수정페이지
- 작성한 내용/값 표시
- 작성 내용/값 검사
- Status Code에 따른 Alert창 띄우기
